Wayweft is a TypeScript-first codebase intelligence layer for AI-assisted development. It helps teams inspect change scope, detect refactor drift, surface structural issues, and package repeatable review workflows into portable agent skills.

## What you get today

- **CLI scanning** — analyze a workspace or package scope for structural drift
- **Multiple report formats** — text, JSON, and SARIF output
- **Safe-fix scaffolding** — low-risk cleanup passes with dry-run previews
- **Skill bundles** — portable workflows for Codex and Claude sessions
